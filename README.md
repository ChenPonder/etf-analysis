# 📈 A股ETF多项式回归分析与预测

基于 **Streamlit** 构建的交互式 Web 应用，对 A 股 ETF 的历史收盘价进行多项式回归分析，并基于统计学准则自动选择最优阶数（1~7 阶），预测未来 7 个交易日的收盘价及 95% 置信区间。

---

## ✨ 核心功能

- **自动识别市场**：用户只需输入纯数字代码（如 `515050`），系统自动补全 `.SH` 或 `.SZ` 后缀。
- **统计学自动选阶**：基于 **F 检验 + BIC 最小化 + 交叉验证（1-SE 法则）**，在 1~7 阶中选出最优多项式阶数，避免过拟合。
- **未来 7 天预测**：输出 t+1 至 t+7 的预测收盘价及 95% 置信区间。
- **交互式图表**：使用 Plotly 绘制实际值、拟合值、预测点及置信区间带状，支持缩放和范围选择。
- **完整统计摘要**：展示 R²、调整 R²、AIC、BIC、各回归系数及其 p 值，以及 1~7 阶对比表。
